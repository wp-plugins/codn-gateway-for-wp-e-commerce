Adds CODN gateway to WP e-Commerce

== Description ==

This is the official CODN gateway plugin for WP e-Commerce.

= How to use it: =
Having installed and activated the plugin, go to the WP e-Commerce settings page and select the Payment tab. Here you'll be able to select and configure the payment gateway.

Once that's done, it will tie in with WP e-Commerce automatically.

= Compatability =
The plugin has been tested with WP e-Commerece 3.9.0.

== Installation ==

1. Upload the `codn-gateway-for-wp-e-commerce` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
